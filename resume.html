📁 Проекты
Анализ и классификация контента Netflix EDA + ML
Python Pandas Matplotlib/Seaborn Scikit-learn
Классификация типа контента Netflix (Movie / TV Show) на основе метаданных. Проведён EDA (9 визуализаций), очистка данных, feature engineering (созданы признаки: content_age, num_cast, title_length, num_genres, month_added). Сравнены 6 моделей, выполнена гиперпараметрическая оптимизация (GridSearchCV) и отбор признаков.
🏆 Лучшая модель: Random Forest (Accuracy = 76.4%, AUC = 0.820)
📊 Ключевые признаки: num_cast (0.187), title_length (0.168), month_added (0.137)
Прогнозирование цен на недвижимость
Python Pandas Statsmodels Scikit-learn
Построение OLS-модели для предсказания цены дома на основе данных kc_house_data. Добавление взаимодействий признаков, анализ значимости через p-value, сравнение R² на train/test.
📈 R² = 0.688, улучшение на +1.72% после добавления взаимодействий
Классификация качества вида дома (Logit vs Probit)
Scikit-learn Statsmodels
Реализация бинарной и мультиномиальной логистической регрессии. Сравнение Logit и Probit моделей по accuracy, log-likelihood и матрице ошибок.
✅ Мультиномиальная модель: 90% точности, Logit и Probit показали сопоставимые результаты
Регуляризация и отбор признаков (Lasso, Ridge, Stepwise)
Scikit-learn NumPy
Сравнение Lasso и Ridge регрессии. На синтетических данных Lasso обнулил 8 из 8 незначимых признаков. Кросс-валидация для выбора оптимальных гиперпараметров.
🔍 Lasso успешно отбирает признаки, Ridge — сжимает коэффициенты
Непараметрическая регрессия (Сплайны, GAM, Gaussian Process)
SciPy PyGAM Scikit-learn
Сравнение 5 методов аппроксимации: полиномы, B-сплайны, сглаживающие сплайны, GAM, Gaussian Process.
🚀 Лучший результат: Gaussian Process (MSE = 0.00278), GAM (MSE = 0.00497)
Сравнение методов оптимизации (DFP, Пауэлл, CEM)
NumPy Matplotlib
Реализация трёх методов для поиска экстремума функции двух переменных. Анализ сходимости из разных начальных точек, визуализация траекторий на контурной карте и в 3D.
⚡ DFP: 6-19 итераций, Пауэлл: стабильный, CEM: глобальный поиск (8000-10000 вызовов)
